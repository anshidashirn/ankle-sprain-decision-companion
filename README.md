# Ankle Sprain Decision Companion

A sophisticated, transparent, and data-driven decision-support system designed to help users evaluate rehabilitation paths for ankle sprains based on personalized medical profiles and priorities.

## üåü Key Features

- **Personalized Injury Profiling**: Input sprain grade (I-III), affected leg, and pain location (Lateral, Medial, High Ankle).
- **Dynamic Decision Logic**: Recommendation scores are automatically adjusted based on the severity of the injury.
- **Weighted Sum Model (WSM)**: Transparent mathematical engine that ranks options based on user-defined priorities (Speed, Pain, Cost, Stability).
- **Personalized Advice**: Specific medical context and tips generated for each rehabilitation path.
- **Glassmorphism UI**: High-fidelity, premium dark mode dashboard built with React and Vite.
