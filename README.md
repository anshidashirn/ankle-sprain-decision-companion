# Ankle Sprain Rehabilitation Decision Companion

## Problem Understanding
This project implements a structured decision-support system that helps users evaluate rehabilitation strategies for ankle sprains.

The goal is not to provide medical advice, but to help structure recovery-related trade-offs.

## Core Features
- Accept multiple rehabilitation options
- Accept weighted criteria
- Compute weighted scores
- Rank options
- Provide explanation for recommendations

## Design Philosophy
- Deterministic logic (no black-box AI decisions)
- Transparent scoring model
- User-adjustable weights
- Explainable recommendations

## Disclaimer
This system is not a medical diagnostic or treatment tool. It is a structured decision-support framework.
